
# Variable Handling

```
  $color:                   black;
  $font_family:             'Helvetica Neue', Arial, Helvetica, sans-serif;
  $base_orange_color:       #D8890E;
  $border:                  1px solid lighten($base_orange_color, 25%);
  $body_color:              #353535;
  $h1_color:                #000;
  $h2_color:                #fff;

  .my_schools {

    color: $body_color;
    border: $border;

    a:hover {
      color: $base_orange_color;
    }
  }
```
